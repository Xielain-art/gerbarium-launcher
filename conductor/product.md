# Initial Concept
Gerbarium Launcher (formerly Helios Launcher) is a custom Minecraft launcher for modded servers that handles account management, efficient asset management, and automatic Java validation.

# Product Definition

## Target Audience
The launcher is tailored for **Casual Players**, focusing on a frictionless, 1-click install experience while keeping complex settings hidden from the default view.

## Core Value Proposition
The launcher acts as a **Community Hub**, prioritizing integrated news feeds, live server status updates, and event announcements to keep players engaged and informed.

## Modpack Management
We follow a **Curated Only** approach, supporting only official, pre-approved server modpacks to guarantee stability and prevent compatibility issues.

## Roadmap Priorities
Future development will focus on a comprehensive set of improvements:
- **Performance:** Optimizing launcher boot times and minimizing the memory footprint.
- **Social Features:** Enhancing Discord RPC integration and exploring in-launcher social tools.
- **Platform Expansion:** Ensuring robust native support across diverse environments like macOS ARM and Linux.
