# Technology Stack

## Core Technologies
- **Language:** TypeScript (Strict Mode)
- **Runtime:** Node.js (Main Process) / Chromium (Renderer Process)
- **Desktop Framework:** Electron

## Frontend Stack
- **Library:** React 19
- **Build Tool/Bundler:** Vite
- **Styling:** Tailwind CSS (v4), shadcn/ui
- **Icons:** Lucide React

## Application State & Data
- **Global State:** Zustand
- **Server State:** TanStack React Query
- **Routing:** TanStack React Router
- **API Interaction:** got, openapi-fetch

## Specialized Libraries
- **Minecraft Core:** helios-core, minecraft-launcher-core
- **Logging:** electron-log
- **Packaging:** electron-builder, esbuild

## Infrastructure & Testing
- **Package Manager:** npm
- **Testing:** Vitest (Unit/Integration), Playwright (Smoke/E2E)
- **Linting:** ESLint
