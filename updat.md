# Обновление

## Для всех пользователей
- Улучшен блок профиля в Dashboard: аватар, роли, переход в настройки и выход теперь выглядят аккуратнее и понятнее.
- Кнопки настроек и выхода переработаны, чтобы ими было проще пользоваться.
- Исправлена проблема на этапе подтверждения кода: кнопка «Войти в другой аккаунт» теперь срабатывает корректно.
- В карточках новостей показываются реальные теги и нормальный анонс, без захардкоженного текста.
- Улучшено отображение changelog: Markdown корректно показывается в окне просмотра.

## Новости (Dashboard)
- Добавлены фильтры новостей:
  - сортировка по дате (сначала новые / сначала старые),
  - фильтр по тегам.
- Удален поиск новостей с Dashboard.
- Исправлен фильтр по тегам (корректная работа с tagId в формате UUID).
- Исправлены стили фильтров и выпадающих списков (больше нет белого фона с плохой читаемостью).
- У тегов в ленте убран символ `#`.

## Changelog
- Добавлено создание changelog в формате Markdown.
- Добавлены редактирование и удаление записей changelog в админке.
- Для просмотра содержимого changelog сделано отдельное модальное окно.
- В списке changelog в админке убран перегруженный предпросмотр контента, оставлены только нужные действия.
- Исправлен баг с горизонтальным скроллом при длинных строках.

## Админ-панель
- Добавлен раздел статистики (Stats) с основными данными.
- В Stats добавлена кнопка «Обновить».
- Убрана глобальная блокирующая загрузка: загрузка теперь показывается локально, только в месте действия.
- Добавлены уведомления через Sonner для понятной обратной связи.
- Улучшен UX создания новостей и changelog-записей.

## Java и производительность
- Улучшен расчет доступной памяти для Java: теперь учитывается реальная оперативная память компьютера.
- Исправлено ограничение, из-за которого на некоторых ПК отображалось только 2 GB.

## Стабильность
- Исправлены ошибки в админских данных и пагинации.
- Улучшена стабильность бесконечной прокрутки в новостях и админских списках.
