# Gerbarium Launcher Product Strategy

## Register
product

## Users
Minecraft players, specifically those playing on modded servers who want a seamless, automated installation and launch experience without manually managing Java versions, Forge, or mod files.

## Product Purpose
Gerbarium Launcher (formerly Helios Launcher) provides a robust, professional-grade desktop application to manage Minecraft accounts, automate the installation of complex modpacks, and provide real-time updates (news, server status) to its community.

## Brand & Tone
- **Professional & Reliable**: The launcher is a tool that should "just work."
- **Developer-Soul**: Inspired by premium developer tools (like Supabase), it feels modern, technical, and high-performance.
- **Direct & Clear**: Communication is high-signal, avoiding fluff.

## Strategic Principles
1. **Performance First**: The UI should feel snappy and efficient.
2. **Visual Hierarchy**: Focus the user's attention on the "Play" button and essential server/account info.
3. **Consistency**: Adhere to the Supabase-inspired design system defined in DESIGN.md.
4. **Information Density**: Provide necessary details (changelogs, news, console logs) without cluttering the primary workflow.

## Anti-References
- Generic, clunky Minecraft launchers with bloated UIs.
- SaaS-cliché "light and airy" designs that lack technical depth.
- Overly gamified interfaces that compromise professional reliability.
